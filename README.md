# react-native-mediapipe

A React Native Camera and Player for MediaPipe applications.

## Installation

```sh
npm install react-native-mediapipe
```

## Usage

```js
import { MediapipeView } from "react-native-mediapipe";

// ...

<MediapipeView color="tomato" />
```
